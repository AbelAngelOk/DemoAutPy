# DemoAutPy
Demo Automation Python

Dependencias Instaladas
- Selenium
- PyTest

Para Ejecutar

Abrir el CMD en el directorio del proyecto y luego ingresar el comando
```
python -m unittest
```

Ingresar a un archivo ./test/*.py y ejecutar el archivo:
```
if __name__ == "__main__":
    unittest.main()
```
